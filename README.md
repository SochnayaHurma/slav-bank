# VAR3 — Fresh modern (slider + silk illustrations)

## Идея
- Оставляем слайдер, но делаем его “современным” за счёт:
  - градиентного слоя (как в дашборде),
  - процедурных SVG-иллюстраций “шелковые штрихи”,
  - мягче жирности и больше читабельности.

## Кнопки
- `.primary` — основной CTA (градиент)
- `.outline` — вторичный CTA
- `.glass` — “стекло” (хорошо для вторичных/шапки/поиска)

## Шрифты (Cygre)
Положи файлы:
- `assets/fonts/Cygre-SemiBold.woff2`
- `assets/fonts/Cygre-Book.woff2`
- `assets/fonts/Cygre-Light.woff2`

## Курсы валют (доверенный источник)
В прототипе нет реальных курсов. В проде лучше подключить официальный источник (например ЦБ РФ).
